[CSS (каскадные таблицы стилей)](https://www.w3.org/Style/CSS/Overview.en.html) — это язык таблиц стилей, который используется вместе с HTML для форматирования и настройки стилей HTML-элементов. Как и в случае с HTML, существует несколько версий CSS, и каждая последующая версия добавляет новый набор возможностей для форматирования HTML-элементов. Браузеры обновляются вместе с выходом новых функций.

---

#### Пример

На базовом уровне CSS используется для определения стиля каждого класса или типа HTML-элементов (например, `body` или `h1`), чтобы любой элемент на странице отображался так, как указано в файле CSS. Это может включать семейство шрифтов, размер шрифта, цвет фона, цвет текста, выравнивание и многое другое.

Код: css

```css
body {
  background-color: black;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: helvetica;
  font-size: 10px;
}
```

Как уже упоминалось, именно поэтому мы можем присваивать уникальным HTML-элементам идентификаторы или имена классов, чтобы при необходимости ссылаться на них в CSS или JavaScript.

---

## Синтаксис

CSS определяет стиль каждого HTML-элемента или класса с помощью фигурных скобок `{}`, внутри которых указываются свойства и их значения (т. е. `element { property : value; }`).

Каждый HTML-элемент имеет множество свойств, которые можно задать с помощью CSS, таких как `height`, `position`, `border` `margin`, `padding` `color`, `text-align` `font-size`, и сотни других свойств. Все это можно объединить и использовать для создания визуально привлекательных веб-страниц.

CSS можно использовать для создания сложных анимаций в самых разных целях: от перемещения элементов до продвинутой 3D-анимации. Для анимации доступно множество свойств CSS, таких как `@keyframes`, `animation`, `animation-duration`, `animation-direction` и многие другие. Вы можете прочитать о многих из этих свойств анимации и опробовать их [здесь](https://www.w3schools.com/css/css3_animations.asp).

#### Использование

CSS часто используется вместе с JavaScript для выполнения быстрых вычислений, динамической настройки свойств стиля определенных HTML-элементов или создания сложных анимаций, реагирующих на нажатия клавиш или положение курсора мыши.

Чтобы просмотреть исходный код, нажмите на вкладки в левом верхнем углу.

Это показывает, что, несмотря на то, что HTML и CSS являются одними из самых базовых инструментов веб-разработки, при правильном использовании их можно применять для создания визуально привлекательных веб-страниц, которые делают взаимодействие с веб-приложениями более простым и удобным.

Кроме того, CSS можно использовать вместе с другими языками для реализации их стилей, например `XML` или внутри `SVG` элементов, а также на современных платформах для разработки мобильных приложений для создания пользовательских интерфейсов (UI) мобильных приложений.

## Фреймворки

Многие считают, что CSS сложен в разработке. Другие же утверждают, что неэффективно вручную задавать стиль и дизайн для всех HTML-элементов на каждой веб-странице. Именно поэтому появилось множество CSS-фреймворков, которые содержат набор таблиц стилей и шаблонов CSS, позволяющих быстрее и проще создавать красивые HTML-элементы.

Кроме того, эти фреймворки оптимизированы для использования в веб-приложениях. Они предназначены для использования с JavaScript и широко применяются в веб-приложениях. Они содержат элементы, которые обычно требуются в современных веб-приложениях. Вот некоторые из наиболее распространённых CSS-фреймворков:

- [Bootstrap](https://www.w3schools.com/bootstrap4/)
- [SASS](https://sass-lang.com/)
- [Foundation](https://en.wikipedia.org/wiki/Foundation_\(framework\))
- [Bulma](https://bulma.io/)
- [Pure](https://purecss.io/)
