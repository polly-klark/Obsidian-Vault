[JavaScript](https://en.wikipedia.org/wiki/JavaScript) — один из самых популярных языков в мире. В основном он используется для веб-разработки и разработки мобильных приложений. `JavaScript` обычно используется во фронтенде приложения, которое запускается в браузере. Тем не менее существуют серверные реализации JavaScript, которые используются для разработки целых веб-приложений, например [NodeJS](https://nodejs.org/en/about/).

В то время как `HTML` и `CSS` в основном отвечают за внешний вид веб-страницы, `JavaScript` обычно используется для управления любой функциональностью, необходимой для внешнего интерфейса веб-страницы. Без `JavaScript` веб-страница была бы в основном статичной и не имела бы особых функций или интерактивных элементов.

---

#### Пример

В исходном коде страницы `JavaScript` код загружается с помощью тега `<script>` следующим образом:

Код: html

```html
<script type="text/javascript">
..JavaScript code..
</script>
```

Веб-страница также может загружать удалённый `JavaScript` код с `src` и ссылкой на скрипт следующим образом:

Код: html

```html
<script src="./script.js"></script>
```

Пример базового использования `JavaScript` на веб-странице:

Код: javascript

```javascript
document.getElementById("button1").innerHTML = "Changed Text!";
```

В приведённом выше примере изменяется содержимое HTML-элемента `button1` . С этого момента `JavaScript` можно использовать на веб-странице гораздо более продвинутым образом. Ниже приведён пример того, что будет делать приведённый выше код `JavaScript` при нажатии на кнопку:

*Оригинальный текст (Нажали на кнопку) Изменнёный текст*

Как и в случае с HTML, в интернете есть множество сайтов, на которых можно поэкспериментировать с `JavaScript`. Один из примеров — [JSFiddle](https://jsfiddle.net/), где можно тестировать `JavaScript`, `CSS`, и `HTML` и сохранять фрагменты кода. `JavaScript` — это продвинутый язык, и его синтаксис не так прост, как `HTML` или `CSS`.

## Использование

Большинство распространённых веб-приложений в значительной степени полагаются на `JavaScript` для реализации всех необходимых функций на веб-странице, таких как обновление веб-страницы в режиме реального времени, динамическое обновление контента в режиме реального времени, приём и обработка пользовательского ввода и многие другие потенциальные функции.

`JavaScript` также используется для автоматизации сложных процессов и выполнения HTTP-запросов для взаимодействия с внутренними компонентами, а также для отправки и получения данных с помощью таких технологий, как [Ajax](https://en.wikipedia.org/wiki/Ajax_\(programming\)).

Помимо автоматизации, `JavaScript` также часто используется вместе с `CSS`, как упоминалось ранее, для создания сложных анимаций, которые были бы невозможны при использовании только `CSS` . Всякий раз, когда мы заходим на интерактивную и динамичную веб-страницу, на которой используется множество сложных и визуально привлекательных анимаций, мы видим результат работы активного кода `JavaScript` в нашем браузере.

Все современные веб-браузеры оснащены `JavaScript` движками, которые могут выполнять `JavaScript` код на стороне клиента, не прибегая к обновлению страницы на внутреннем веб-сервере. Это позволяет очень быстро выполнять большое количество процессов с помощью `JavaScript`

## Фреймворки

По мере развития веб-приложений использование чистого `JavaScript` для разработки всего веб-приложения с нуля может оказаться неэффективным. Именно поэтому появилось множество `JavaScript` фреймворков, которые упрощают разработку веб-приложений.

Эти платформы предоставляют библиотеки, которые значительно упрощают создание расширенных функций, таких как вход в систему и регистрация пользователей, а также внедряют новые технологии на основе уже существующих, например использование динамически изменяющегося `HTML`-кода вместо статического `HTML`-кода.

Эти платформы либо используют `JavaScript` в качестве языка программирования, либо используют реализацию `JavaScript`, которая компилирует код в `JavaScript`-код.

Одними из самых распространённых интерфейсных `JavaScript`-фреймворков являются:

- [Angular](https://www.w3schools.com/angular/angular_intro.asp)
- [React](https://www.w3schools.com/react/react_intro.asp)
- [Vue](https://www.w3schools.com/whatis/whatis_vue.asp)
- [jQuery](https://www.w3schools.com/jquery/)

Список и сравнение распространённых JavaScript-фреймворков можно найти [здесь](https://en.wikipedia.org/wiki/Comparison_of_JavaScript_frameworks).
